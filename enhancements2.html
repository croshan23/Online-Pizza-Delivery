Enhancements Used

  1. Preloading Full Name on Card Holder Name
    • When we need a client to return and refresh information through a shape, we have to "preload" the information they entered beforehand, as a rule from the database. In preloading textboxes and textareas we stack information into the esteem property of the shape component.
    • Preloading data in forms removes work load for users to enter same information again and again.
    • I have used preloading card holder name in the payment form that is in "payment.html" page, but to go there first we need to place the order through enquire.html page then it will redirect you to payment.html page where you can see card holder name preloaded. To check Click Here
  2. Dynamically Load Products using Java Script Variables
    • Loading data dynamically is generally used in enterprise application where data in web page is needed to be downloaded from server, which then is displayed dynamically for users.
    • Using this techniques we can develop dynamic website where contents are not hard coded in the page.
    • I have used to dynamically load products name in product.html using javaScript. First names are stored in Variables and is displayed dynamically during loading of the page. To check Click Here
